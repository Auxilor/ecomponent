## Get from JitPack:

Gradle:

```groovy
repositories {
    maven { url 'https://jitpack.io' }
}

```

```groovy
dependencies {
    compileOnly 'com.willfp:ecomponent:Tag'
}
```

Replace `Tag` with a release tag for ecomponent, eg `1.0.0`.
